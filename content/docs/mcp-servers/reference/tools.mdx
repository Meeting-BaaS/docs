# MCP Tools Reference

This page provides technical reference documentation for the tools available in Meeting BaaS MCP servers.

## Meeting Management Tools

<Accordions>
  <Accordion title="createBot" icon={<Plus />} defaultOpen>
    Creates a meeting bot that can join video conferences to record and transcribe meetings.
    
    ### Parameters
    
    | Parameter | Type | Description | Required |
    | --- | --- | --- | --- |
    | `meeting_url` | string | URL of the meeting to join | Yes |
    | `name` | string | Bot name | No |
    | `botImage` | string | URL to an image for the bot's avatar | No |
    | `entryMessage` | string | Message the bot will send when joining | No |
    | `deduplicationKey` | string | Key to override the 5-minute restriction on joining the same meeting | No |
    | `nooneJoinedTimeout` | number | Timeout in seconds for bot to leave if no one joins | No |
    | `waitingRoomTimeout` | number | Timeout in seconds for bot to leave if stuck in waiting room | No |
    | `speechToTextProvider` | string | Provider for transcription: "Gladia", "Runpod", or "Default" | No |
    | `speechToTextApiKey` | string | API key for the speech-to-text provider | No |
    | `extra` | object | Additional metadata about the meeting | No |
    
    ### Returns
    
    Bot details including ID and join status.
    
    ### Example
    
    ```json
    {
      "meeting_url": "https://meet.google.com/abc-def-ghi",
      "name": "Meeting Assistant",
      "botImage": "https://example.com/bot.jpg",
      "entryMessage": "Hello, I'm here to record the meeting."
    }
    ```
  </Accordion>
  
  <Accordion title="getBots" icon={<List />}>
    Lists all bots and their associated meetings.
    
    ### Parameters
    
    None
    
    ### Returns
    
    Array of bot objects with details about each bot.
    
    ### Example
    
    ```json
    []
    ```
  </Accordion>
  
  <Accordion title="getBotsByMeeting" icon={<Search />}>
    Gets bots for a specific meeting URL.
    
    ### Parameters
    
    | Parameter | Type | Description | Required |
    | --- | --- | --- | --- |
    | `meetingUrl` | string | URL of the meeting to get bots for | Yes |
    
    ### Returns
    
    Array of bot objects associated with the specified meeting URL.
    
    ### Example
    
    ```json
    {
      "meetingUrl": "https://meet.google.com/abc-def-ghi"
    }
    ```
  </Accordion>
  
  <Accordion title="getMeetingData" icon={<FileText />}>
    Gets transcript and recording data for a specific meeting.
    
    ### Parameters
    
    | Parameter | Type | Description | Required |
    | --- | --- | --- | --- |
    | `meetingId` | string | ID of the meeting to get data for | Yes |
    
    ### Returns
    
    Information about the meeting recording including duration and transcript segment count.
    
    ### Example
    
    ```json
    {
      "meetingId": "67738f48-2360-4f9e-a999-275a74208ff5"
    }
    ```
  </Accordion>
</Accordions>

## Calendar Integration Tools

<Accordions>
  <Accordion title="oauthGuidance" icon={<HelpCircle />} defaultOpen>
    Get detailed step-by-step instructions on setting up OAuth for Google or Microsoft calendars.
    
    ### Parameters
    
    None
    
    ### Returns
    
    Comprehensive instructions for obtaining OAuth credentials and setting up calendar integration.
  </Accordion>
  
  <Accordion title="setupCalendarOAuth" icon={<Calendar />}>
    Integrates a calendar using OAuth credentials.
    
    ### Parameters
    
    | Parameter | Type | Description | Required |
    | --- | --- | --- | --- |
    | `platform` | string | "Google" or "Microsoft" | Yes |
    | `clientId` | string | OAuth client ID | Yes |
    | `clientSecret` | string | OAuth client secret | Yes |
    | `refreshToken` | string | OAuth refresh token | Yes |
    | `rawCalendarId` | string | ID of the specific calendar to integrate | No |
    
    ### Returns
    
    Confirmation of successful integration with calendar details.
    
    ### Example
    
    ```json
    {
      "platform": "Google",
      "clientId": "client-id-123456789.apps.googleusercontent.com",
      "clientSecret": "client-secret-ABCDEF123456",
      "refreshToken": "refresh-token-ABCDEF123456789",
      "rawCalendarId": "primary@gmail.com"
    }
    ```
  </Accordion>
  
  <Accordion title="listEvents" icon={<List />}>
    Lists calendar events with comprehensive filtering options.
    
    ### Parameters
    
    | Parameter | Type | Description | Required |
    | --- | --- | --- | --- |
    | `calendarId` | string | UUID of the calendar | Yes |
    | `attendeeEmail` | string | Filter by attendee email | No |
    | `startDateGte` | string | Filter by start date >= (ISO format) | No |
    | `startDateLte` | string | Filter by start date <= (ISO format) | No |
    | `status` | string | Filter by event status | No |
    
    ### Returns
    
    Detailed event listings with rich information.
    
    ### Example
    
    ```json
    {
      "calendarId": "1a2b3c4d-5e6f-7a8b-9c0d-1e2f3a4b5c6d",
      "startDateGte": "2023-12-01T00:00:00Z",
      "startDateLte": "2023-12-31T23:59:59Z"
    }
    ```
  </Accordion>
  
  <Accordion title="scheduleRecording" icon={<Video />}>
    Schedules a bot to record an upcoming meeting.
    
    ### Parameters
    
    | Parameter | Type | Description | Required |
    | --- | --- | --- | --- |
    | `eventId` | string | UUID of the event | Yes |
    | `botName` | string | Name for the bot | Yes |
    | `botImage` | string | URL to an image for the bot's avatar | No |
    | `recordingMode` | string | Mode of recording (e.g., "gallery_view") | No |
    | `entryMessage` | string | Message the bot will send when joining | No |
    
    ### Returns
    
    Confirmation of successful scheduling.
    
    ### Example
    
    ```json
    {
      "eventId": "7a8b9c0d-1e2f-3a4b-5c6d-7e8f9a0b1c2d",
      "botName": "Team Meeting Bot",
      "recordingMode": "gallery_view"
    }
    ```
  </Accordion>
</Accordions>

## Transcript Tools

<Accordions>
  <Accordion title="getMeetingTranscript" icon={<FileText />} defaultOpen>
    Gets a meeting transcript with speaker names and content grouped by speaker.
    
    ### Parameters
    
    | Parameter | Type | Description | Required |
    | --- | --- | --- | --- |
    | `botId` | string | ID of the bot that recorded the meeting | Yes |
    
    ### Returns
    
    Complete transcript with speaker information, formatted as paragraphs grouped by speaker.
    
    ### Example
    
    ```json
    {
      "botId": "abc-123"
    }
    ```
  </Accordion>
  
  <Accordion title="findKeyMoments" icon={<Star />}>
    Automatically identifies and shares links to important moments in a meeting.
    
    ### Parameters
    
    | Parameter | Type | Description | Required |
    | --- | --- | --- | --- |
    | `botId` | string | ID of the bot that recorded the meeting | Yes |
    | `meetingTitle` | string | Title of the meeting | No |
    | `topics` | array | List of topics to look for | No |
    | `maxMoments` | number | Maximum number of moments to identify | No |
    
    ### Returns
    
    Markdown-formatted list of key moments with links, automatically detected based on transcript.
    
    ### Example
    
    ```json
    {
      "botId": "xyz-456",
      "topics": ["budget", "timeline", "deliverables"],
      "maxMoments": 5
    }
    ```
  </Accordion>
</Accordions>

## QR Code Tools

<Accordions>
  <Accordion title="generateQRCode" icon={<QrCode />} defaultOpen>
    Creates an AI-generated QR code image that can be used as a bot avatar.
    
    ### Parameters
    
    | Parameter | Type | Description | Required |
    | --- | --- | --- | --- |
    | `type` | string | Type of QR code (url, email, phone, sms, text) | Yes |
    | `to` | string | Destination for the QR code (URL, email, phone number, or text) | Yes |
    | `prompt` | string | AI prompt to customize the QR code | Yes |
    | `style` | string | Style of the QR code (style_default, style_dots, style_rounded, style_crystal) | No |
    | `useAsBotImage` | boolean | Whether to use the generated QR code as the bot avatar | No |
    | `apiKey` | string | Your QR Code AI API key | No |
    
    ### Returns
    
    URL to the generated QR code image that can be used directly with the createBot tool.
    
    ### Example
    
    ```json
    {
      "type": "email",
      "to": "john.doe@example.com",
      "prompt": "Create a QR code that looks like a mountain landscape",
      "style": "style_rounded"
    }
    ```
  </Accordion>
</Accordions>

## Link Sharing Tools

<Accordions>
  <Accordion title="shareableMeetingLink" icon={<Link />} defaultOpen>
    Generates a nicely formatted, shareable link to a meeting recording.
    
    ### Parameters
    
    | Parameter | Type | Description | Required |
    | --- | --- | --- | --- |
    | `botId` | string | ID of the bot that recorded the meeting | Yes |
    | `timestamp` | number | Timestamp in seconds to link to | No |
    | `title` | string | Title for the shared link | No |
    | `speakerName` | string | Name of the speaker at the timestamp | No |
    | `description` | string | Description of what's happening at the timestamp | No |
    
    ### Returns
    
    Markdown-formatted link with metadata that can be shared directly in chat.
    
    ### Example
    
    ```json
    {
      "botId": "abc-123",
      "timestamp": 755,
      "title": "Weekly Team Sync",
      "speakerName": "Sarah Johnson",
      "description": "Discussing the new product roadmap"
    }
    ```
  </Accordion>
  
  <Accordion title="shareMeetingSegments" icon={<ListOrdered />}>
    Creates a list of links to multiple important moments in a meeting.
    
    ### Parameters
    
    | Parameter | Type | Description | Required |
    | --- | --- | --- | --- |
    | `botId` | string | ID of the bot that recorded the meeting | Yes |
    | `segments` | array | Array of segments with timestamps, speakers, and descriptions | Yes |
    
    ### Returns
    
    Markdown-formatted list of segments with direct links to each moment.
    
    ### Example
    
    ```json
    {
      "botId": "abc-123",
      "segments": [
        {
          "timestamp": 120,
          "speakerName": "John Smith",
          "description": "Introduction and agenda"
        },
        {
          "timestamp": 450,
          "speakerName": "Sarah Johnson",
          "description": "Q2 sales results"
        }
      ]
    }
    ```
  </Accordion>
</Accordions>
