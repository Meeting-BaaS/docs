    ```text
    I pushed a fix in commit <commit_id>, please review it.

    ```
  </Tab>

  <Tab value="File Comments">
    Under "Files changed", tag @rabbitformeetingbaas in a new review comment:

    ```text
    @rabbitformeetingbaas generate unit testing code for this file.
    @rabbitformeetingbaas modularize this function.
    ```
  </Tab>

  <Tab value="PR Comments">
    Tag @rabbitformeetingbaas in PR comments for branch-wide queries:

    ```text
    @rabbitformeetingbaas read src/utils.ts and generate unit testing code.
    @rabbitformeetingbaas help me debug CodeRabbit configuration file.
    ```
  </Tab>
</Tabs>

## Available Commands

<Accordions>
  <Accordion title="PR Commands" value="pr-commands">
    - `@rabbitformeetingbaas pause` - Pause PR reviews
    - `@rabbitformeetingbaas resume` - Resume reviews
    - `@rabbitformeetingbaas review` - Trigger incremental review
    - `@rabbitformeetingbaas full review` - Complete review from scratch
    - `@rabbitformeetingbaas summary` - Regenerate PR summary
    - `@rabbitformeetingbaas generate sequence diagram` - Create sequence diagram
    - `@rabbitformeetingbaas resolve` - Resolve all review comments
    - `@rabbitformeetingbaas configuration` - Show current config
    - `@rabbitformeetingbaas help` - Get assistance
  </Accordion>

  <Accordion title="Special Keywords" value="keywords">
    - `@rabbitformeetingbaas ignore` in PR description - Skip review
    - `@rabbitformeetingbaas summary` or `@coderabbitai summary` - Generate summary
    - `@rabbitformeetingbaas` or `@coderabbitai` in PR title - Auto-generate title
  </Accordion>
</Accordions>
