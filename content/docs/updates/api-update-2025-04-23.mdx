# API Updates <span className="text-sm text-muted-foreground ml-2">April 23rd, 2025</span>

## Bot Endpoints

### GET /bots/with/metadata

- Streamlined response format
- All filtering capabilities preserved
- Advanced filtering/sorting with `filter_by_extra` and `sort_by_extra`
- Returns essential metadata (IDs, names, meeting details)

### GET /bots/:uuid/screenshots

- New endpoint for bot session screenshots
- Useful for monitoring and verification
- Part of expanded bot monitoring capabilities

## Webhook Endpoints

### GET /webhooks/calendar/webhook/documentation

- Parameter notation updated from `{uuid}` to `:uuid` for REST API consistency
- Clarified `affected_event_uuids` field documentation
- Enhanced calendar event integration examples
- All endpoints now use `:parameter` notation

These changes will be live in production on April 24th, 2025.
