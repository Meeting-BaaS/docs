# Chatting with CodeRabbit üêü

*Something's fishy here... but in a good way! Let's dive into how to communicate with CodeRabbit - your AI code review companion that's always ready to take the bait! üé£*

<Tabs items={['Review Comments', 'Code Files', 'PR Comments']}>
  <Tab value="Review Comments">
    Simply reply directly to CodeRabbit's review comments with queries like:

    ```text
    I pushed a fix in commit abc123, please review it.
    Explain this complex logic.
    ```
  </Tab>

  <Tab value="Code Files">
    Under the "Files changed" tab, tag @rabbitformeetingbaas in a new review comment:

    ```text
    @rabbitformeetingbaas explain this code block.
    @rabbitformeetingbaas modularize this function.
    ```
  </Tab>

  <Tab value="PR Comments">
    Tag @rabbitformeetingbaas in PR comments for broader questions:

    ```text
    @rabbitformeetingbaas gather interesting stats about this repository
    @rabbitformeetingbaas read src/utils.ts and explain its purpose
    ```
  </Tab>
</Tabs>

<Callout type="info">
  For best results, be specific with your queries and break down large tasks into smaller chunks. Review comments are ideal for focused discussions about specific code changes.
</Callout>

## Available Commands

<Accordions>
  <Accordion title="PR Review Commands" value="review-commands">
    - `@rabbitformeetingbaas review` - Trigger incremental review
    - `@rabbitformeetingbaas full review` - Perform complete review
    - `@rabbitformeetingbaas pause/resume` - Control review process
    - `@rabbitformeetingbaas summary` - Regenerate PR summary
    - `@rabbitformeetingbaas resolve` - Resolve all review comments
  </Accordion>

  <Accordion title="Special Keywords" value="keywords">
    - `@rabbitformeetingbaas ignore` in PR description - Skip review
    - `@rabbitformeetingbaas summary` in PR description - Generate summary
    - `@rabbitformeetingbaas` in PR title - Auto-generate title
  </Accordion>
